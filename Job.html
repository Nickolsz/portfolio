Job Application Tracker

To serve as an alternative to my Email-To-Sankey project, I created a simple Microsoft Form that a user can answer after finishing a job application. It asks simple, but important questions that could not be tracked in the other project. This allows for a more personal assessment of the jobs that they are applying to. This more personal assessment will allow the user to change up their job application strategy if they find more success in one week compared to another.

Once the form is submitted, the answers are loaded into a Microsoft Sheet. This serves as our pseudo-database. From here the data is automatically sent to the PowerBI Dashboard. In this dashboard there is also a calendar database that has a relationship with the form response table to account for days where no applications were sent out. On top of this it allows our dashboard to be organized by week. The dashboard updates every week and, my next step of this project is to implement a Power Automate feature that will capture a screenshot, or export the dashboard before the new week is created.

Everything in the project was created by me, from the form, connections, and the PowerBI Template/Dashboard.
